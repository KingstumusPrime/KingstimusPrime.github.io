Custom OS

A custom POSIX compatible 32 bit operating and C Library written from complete scratch. Written primarily in C but leverages assembly to complete low level tasks and boot the operating system. Contains a monolithic kernel design and supports the FAT filesystem, ATA drives, basic I/O, user mode, virtual memory, VESA video modes, and Linux-style file handling. Along with this it has a relatively mature LibC and operating systems specific toolchain capable of porting basic programs such as the famous Doom I & II.

NES Emulator

A working NES emulator written from complete scratch using C++ and SDL2 for the rendering and keyboard input. Includes a full 6502 emulator, a cycle accurate Picture Processing Unit (PPU), and a couple of debug tools such as a basic 6502 disassembler. Uses a proper object-oriented approach to emulate every single component as its own object making the codebase both easy to debug and conceptually correct.

Crown.JS

Crown.JS is a Javascript library that attempts to streamline the creation of canvas based games in Javascript. It leverages Inherence and a classical object-oriented structure to allow the user to expand upon the engine and add features needed only for that project. Features include sprites, actors, a custom input system, a smooth movement library, and dynamic camera that supports shake. In addition the project also contains a bonus multiplayer header that serves as a wrapper for PEER.JS. Allowing for multiplayer functionality in minutes.
